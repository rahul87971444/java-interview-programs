# 🔁 Palindrome String in Java

A palindrome is a string that reads the same forward and backward.

---

# 📌 Problem Statement

Write a Java program to check whether a given string is a palindrome.

---

## Example 1

Input

```
madam
```

Output

```
Palindrome
```

---

## Example 2

Input

```
java
```

Output

```
Not Palindrome
```

# 🚀 Approaches

| Approach | Time | Space |
|----------|------|-------|
| Reverse String | O(n) | O(n) |
| Two Pointer | O(n) | O(1) |

---

# 🧠 Algorithm (Reverse String)

1. Read the string.
2. Reverse it.
3. Compare original and reversed strings.
4. If equal → Palindrome.

---

# ⚡ Algorithm (Two Pointer)

1. Place one pointer at the beginning.
2. Place another pointer at the end.
3. Compare both characters.
4. If any mismatch occurs → Not Palindrome.
5. Otherwise continue until pointers meet.

---

# 🌍 Real-Life Example

```
MADAM
```

Read from left to right

```
MADAM
```

Read from right to left

```
MADAM
```

Same in both directions.
