# 🔤 Count Vowels in Java

Learn how to count vowels in a string using Java with multiple approaches.

---

# 📌 Problem Statement

Write a Java program to count the number of vowels present in a string.

Vowels are:

```
a
e
i
o
u
```

---

## Example

Input

```
Education
```

Output

```
5
```

Explanation

```
E
u
a
i
o
```

---

# 🎯 Learning Objectives

After completing this program you will learn

- String Traversal
- charAt()
- Character Comparison
- Character Case Conversion
- Time Complexity

---

# 🚀 Approaches

| Approach | Time | Space |
|----------|------|-------|
| Simple Loop | O(n) | O(1) |
| toLowerCase() | O(n) | O(1) |

---

# 🧠 Algorithm

1. Read the string.
2. Convert it into lowercase.
3. Traverse every character.
4. Check whether it is

```
a
e
i
o
u
```

5. Increase count.
