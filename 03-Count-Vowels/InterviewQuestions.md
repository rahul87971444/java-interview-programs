# Count Vowels - Interview Questions

## Q1. Which method returns a character?

```
charAt()
```

---

## Q2. Why use toLowerCase()?

To avoid checking both

```
A

a
```

---

## Q3. Time Complexity?

```
O(n)
```

---

## Q4. Space Complexity?

```
O(1)
```

---

## Q5. Difference between charAt() and toCharArray()?

charAt()

Returns one character using index.

toCharArray()

Converts the whole string into a character array.
