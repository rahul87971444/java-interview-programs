# 🚀 Remove Spaces from a String in Java

Learn how to remove spaces from a string using Java with multiple approaches.

---

# 📌 Problem Statement

Write a Java program to remove all spaces from a given string.

---

## Example

### Input

```
Java Interview Preparation
```

### Output

```
JavaInterviewPreparation
```

---

# 🎯 Learning Objectives

- String Manipulation
- replace()
- replaceAll()
- StringBuilder
- Regular Expressions (Regex)

---

# 🚀 Approaches

| Approach | Time | Space |
|----------|------|-------|
| replace() | O(n) | O(n) |
| StringBuilder | O(n) | O(n) |

---

# 🧠 Algorithm

1. Read the string.
2. Traverse every character.
3. Ignore spaces.
4. Store remaining characters.
5. Print the result.

---

# 🌍 Real-Life Example

Suppose a user enters:

```
John Doe
```

Many applications remove spaces before generating usernames.

Output

```
JohnDoe
```
