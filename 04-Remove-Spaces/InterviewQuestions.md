# Remove Spaces - Interview Questions

## Q1. Which method removes spaces?

```
replace()
```

---

## Q2. Difference between replace() and replaceAll()?

replace()

- Replaces exact characters.

replaceAll()

- Uses Regular Expressions (Regex).

---

## Q3. Why use StringBuilder?

Because Strings are immutable and StringBuilder is faster for repeated modifications.

---

## Q4. Complexity?

Time

```
O(n)
```

Space

```
O(n)
```

---

## Q5. Can we remove only leading and trailing spaces?

Yes.

```
trim()
```
