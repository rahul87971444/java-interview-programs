# 🔄 Reverse String in Java

Reverse a string using different approaches in Java while understanding important interview concepts such as **String Immutability**, **Two Pointers**, and **StringBuilder**.

---

# 📌 Problem Statement

Given a string, reverse it without using any built-in reverse method.

### Example

**Input**

```
Rahul
```

**Output**

```
luhaR
```

---

---

# 🚀 Approaches

| Approach | Time | Space |
|----------|------|-------|
| Loop | O(n) | O(n) |
| Two Pointer | O(n) | O(n) |
| StringBuilder | O(n) | O(n) |
| Recursion | O(n) | O(n) |

---

# 🧠 Algorithm (Loop)

1. Read the string.
2. Start from the last character.
3. Append each character to a new string.
4. Print the reversed string.

---

# ⚡ Algorithm (Two Pointer)

1. Convert the string into a character array.
2. Place one pointer at the beginning and another at the end.
3. Swap both characters.
4. Move the pointers toward the center.
5. Convert the array back into a string.

---

# ⏱ Complexity

| Approach | Time | Space |
|----------|------|-------|
| Loop | O(n) | O(n) |
| Two Pointer | O(n) | O(n) |

# 🌍 Real-Life Example

Imagine reading a word written on paper from the last letter to the first.

```
Original

R a h u l
```

Reading backwards

```
l u h a R
```

That is exactly how the reverse string algorithm works.
